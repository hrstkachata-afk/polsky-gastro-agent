# WhatsApp zprávy pro Moniku

WhatsApp používat jen na důležité věci. Jinak začne být systém otravný.

## Nový důležitý kontakt

```text
Moniko, nový kontakt čeká na schválení:
{{company_name}} - {{city}}
Typ: {{contact_type}}

Draft je připravený v Gmailu a karta je v Trellu.
```

## Odpověď se zájmem

```text
Moniko, přišla odpověď se zájmem z Polska.

Kontakt: {{company_name}}
Shrnutí: {{summary_cs}}
Počet osob: {{people_count}}
Termín: {{requested_date}}

Další krok: {{next_step_cs}}
V Gmailu je připravený draft odpovědi.
```

## Chtějí termín

```text
Moniko, kontakt z Polska chce řešit termín.

Kontakt: {{company_name}}
Termín: {{requested_date}}
Počet osob: {{people_count}}

Podívej se prosím do Trella a potvrď možnosti.
```

## Chtějí cenu

```text
Moniko, kontakt chce cenu nabídky.

Kontakt: {{company_name}}
Shrnutí: {{summary_cs}}

Připravil se draft odpovědi v Gmailu. Cena je zatím orientačně kolem 600 Kč/os.
```
